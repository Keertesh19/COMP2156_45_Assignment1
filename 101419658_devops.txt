COMP 2156 is a DevOps course offered at George Brown College, aiming to provide students with a comprehensive understanding of DevOps principles and practices.
Covering essential topics such as version control systems, continuous integration and deployment, configuration management, containerization, and infrastructure as code, the course equips students with the skills needed to streamline collaboration and communication between development and IT teams.
With a focus on hands-on projects, students have the opportunity to apply DevOps concepts in practical scenarios, preparing them for the dynamic and collaborative environment of modern software development and deployment.

Key Topics Covered:

Introduction to DevOps and its principles.
Version control systems (e.g., Git).
Continuous Integration (CI) and Continuous Deployment (CD) pipelines.
Configuration management tools (e.g., Ansible, Chef, Puppet).
Containerization and orchestration (e.g., Docker, Kubernetes).
Infrastructure as Code (IaC) concepts.
Monitoring and logging in a DevOps environment.
Collaboration and communication tools.
